# Contributing to Caremi

Thank you for your interest in contributing! We welcome all kinds of contributions — from code to documentation and everything in between.

---

## 💡 How to Contribute

### 1. Fork the Repository

Click the **Fork** button in the top-right of [caremillc/caremi-pro-mvc](https://github.com/caremillc/caremi-pro-mvc) and clone your fork:

```bash
git clone https://github.com/caremillc/caremi-pro-mvc.git caremi

cd caremi
```
